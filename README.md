# LL(1) Parser (GUI-Based)

A **GUI-based LL(1) predictive parser** that lets you input a context-free grammar, computes **FIRST/FOLLOW**, builds the **LL(1) parsing table**, and parses a tokenised input string with a step-by-step trace and parse tree visualisation. 

## Features
- Grammar input: `A -> α | β` (space-separated symbols)
- `#` for epsilon, `$` as end marker
- FIRST/FOLLOW computation
- LL(1) parsing table generation (conflict detection)
- Parsing trace (stack + input buffer steps)
- Parse tree output + GUI display

## Requirements
- Python 3.x
- Tkinter (usually included with Python on Windows)
